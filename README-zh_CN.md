## 简介
该项目旨在服务器上生成自定义 PAC 文件，以改善移动设备上的代理设置体验
## 技术栈
[Node.js] | [Egg.js]

## 快速开始



```bash
开发流程
$ npm i
$ npm run dev
$ open http://localhost:7001/ by browsers

部署流程
$ npm start
$ npm stop
```
